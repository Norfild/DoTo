Default grid

The default grid system provided as part of Bootstrap is a 940px wide 16-column grid. Itâ€™s a flavor of the popular 960 grid system, but without the additional margin/padding on the left and right sides.

Example grid markup

As shown here, a basic layout can be created with two "columns," each spanning a number of the 16 foundational columns we defined as part of our grid system. See the examples below for more variations.

  <div class="span6 columns">
    ...
  </div>
